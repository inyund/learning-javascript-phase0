# 📝 Bab 01: Variables & Data Types Mutability

## 🎯 Objective
1. Memahami pembuatan variabel dengan kata kunci `const`, `let`, dan `var`.
2. Memahami sifat *primitive immutability* pada String vs *reference mutability* pada Array.
3. Melatih pengecekan tipe data menggunakan operator `typeof` serta penanganan kondisi *falsy*.

---

## 📂 Latihan 1: Mutabilitas Variabel (`mutability-check.md` & `mutability-check.js`)
Cobalah ubah nilai indeks karakter dari sebuah String dan elemen dari sebuah Array. Amati mana yang nilainya dapat berubah dan jelaskan alasannya pada file jawaban.

## 📂 Latihan 2: Pengecekan Tipe Data (`data-type-check.md` & `data-type-check.js`)
Buat pengkondisian sederhana untuk memeriksa tipe data dari variabel `kata`:
- Jika tipe data = `string` ➡️ Tampilkan `username [kata]`
- Jika tipe data = `number` ➡️ Tampilkan `age [kata]`
- Jika tipe data = `boolean` ➡️ Tampilkan respon persetujuan
- Jika variabel bernilai kosong/falsy ➡️ Tampilkan `Invalid Data`
