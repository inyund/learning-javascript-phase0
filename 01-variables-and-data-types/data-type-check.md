# 🔍 Pengecekan Tipe Data (typeof)

## Objective
Mampu menguji dan mengidentifikasi tipe data dinamis menggunakan kata kunci `typeof` serta penanganan kondisi *falsy*.

## Problem Statement
Buatlah logika pengkondisian untuk mengecek tipe data dari variabel `kata`:
- Jika tipe datanya adalah `string`, tampilkan: `username [kata]`
- Jika tipe datanya adalah `number`, tampilkan: `age [kata]`
- Jika tipe datanya adalah `boolean`:
  - Jika bernilai `true`, tampilkan: `thank you for agreeing`
  - Jika bernilai `false`, tampilkan: `cannot proceed without agreement`
- Jika variabel `kata` bernilai kosong/falsy/null/undefined, tampilkan: `Invalid Data`
