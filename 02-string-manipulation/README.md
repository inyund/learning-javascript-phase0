# 🔤 Bab 02: String Manipulation & Indexing

## 🎯 Objective
1. Mengakses karakter String berdasarkan indeks posisi (`str[i]`).
2. Menggabungkan beberapa string (*concatenation*).
3. Mengambil potongan string dengan metode bawaan `substring()`.
4. Membandingkan dua string & membalikkan string secara manual.
