# ⚙️ Challenge: Basic Functions & Return Values

## Objective
Memahami deklarasi fungsi, pemanggilan fungsi dengan parameter, penggunaan `return`, dan penggabungan antar-fungsi.

## Problem Statement
1. Buat fungsi `tambah(valueA, valueB)` yang mengembalikan hasil penjumlahan `valueA + valueB` ditambah dengan variabel luar `bullcrap = 5`.
2. Buat fungsi `perkalian(valueA, valueB, valueC)` yang memanfaatkan hasil pemanggilan fungsi `tambah(valueA, valueC)`.
