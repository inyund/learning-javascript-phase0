# ❌⭕ Challenge: X and O Counter

## Objective
Menghitung frekuensi karakter pada string dan membandingkan kesamaannya.

## Problem Statement
Diberikan sebuah string yang berisi campuran huruf `x`, `o`, dan huruf lainnya:
```javascript
var kata = 'xoxoasdasdasdasdasdasdasxo';
```
Hitung jumlah kemunculan huruf `x` dan huruf `o`. Tampilkan `true` jika jumlah `x` sama dengan `o`, dan tampilkan `false` jika tidak sama.
