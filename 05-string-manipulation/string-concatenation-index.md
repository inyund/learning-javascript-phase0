# 🔤 Challenge: Play With Strings

## Objective
Memahami penggabungan string (*concatenation*), pengaksesan karakter via indeks (`str[i]`), substring (`substring()`), dan menghitung panjang string (`.length`).

## Problem Statement
1. Gabungkan variabel kata-kata yang tersedia menjadi satu kalimat utuh dengan spasi.
2. Ambil kata per kata dari kalimat `"wow JavaScript is so cool"` menggunakan teknik pengaksesan indeks karakter (`word2[0] + ...`).
3. Ambil kata per kata dari kalimat `"wow JavaScript is so cool"` menggunakan metode `substring()`, lalu hitung dan tampilkan panjang karakter dari masing-masing kata tersebut.
