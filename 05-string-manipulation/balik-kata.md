# 🔄 Challenge: Balik Kata

## Objective
Membalikkan urutan karakter pada sebuah string dari belakang ke depan secara manual menggunakan loop.

## Problem Statement
Diberikan variabel `kata = "Javascript"`.
Buatlah logika untuk membalikkan string tersebut sehingga menghasilkan output `"tpircsavaJ"`.

*Catatan: Tidak boleh menggunakan fungsi bawaan `reverse()`.*
