# 鈿栵笍 Challenge: Bandingkan Angka

## Objective
Membandingkan dua buah angka dan mengembalikan nilai boolean atau nilai khusus.

## Problem Statement
Diberikan `angka1` dan `angka2`:
- Jika `angka2` lebih besar dari `angka1`, tampilkan `true`.
- Jika `angka1` lebih besar dari `angka2`, tampilkan `false`.
- Jika kedua angka sama besar, tampilkan `"-1"`.
