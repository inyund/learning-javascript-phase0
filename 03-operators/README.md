# ➕ Bab 03: Operators

Operator adalah simbol yang memproses nilai — menjumlah, membandingkan, menggabungkan logika. Konsep ini dipakai di **hampir semua** soal bab berikutnya (conditionals, loops, functions), jadi kuasai dulu di sini.

## 📌 Urutan Pengerjaan Soal

1. **`03-1-arithmetic-basics.md`** ➡️ `+ - * / % **` dan penggabungan string.
2. **`03-2-comparison-and-equality.md`** ➡️ `>` `<` `>=` `<=`, serta `==` vs `===`.
3. **`03-3-logical-operators.md`** ➡️ `&&`, `||`, `!`.
4. **`03-4-compound-assignment.md`** ➡️ `+= -= *=` dan `++ --`.

> 📎 Operator ternary (`kondisi ? a : b`) sengaja **tidak** di sini — karena dia bentuk singkat `if/else`, kamu akan mempelajarinya di Bab 04 (Conditionals) setelah paham `if/else` dulu.

## 🛠️ Cara Menjawab
Tulis kodemu di file `.js` yang bernama sama dengan soalnya, lalu jalankan dengan `node <nama-file>.js`.
