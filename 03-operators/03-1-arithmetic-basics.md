# ➗ 03-1: Arithmetic Basics

## 🎯 Konsep yang Dilatih
Operator aritmatika: `+` (tambah), `-` (kurang), `*` (kali), `/` (bagi), `%` (sisa bagi / modulus), `**` (pangkat). Plus peran ganda `+` untuk **menggabung string**.

## ✅ Tugas
Diberikan `let a = 17` dan `let b = 5`. Cetak hasil dari:
1. `a + b`
2. `a - b`
3. `a * b`
4. `a / b`
5. `a % b`  *(sisa pembagian)*
6. `a ** 2`  *(a pangkat 2)*

Lalu, dengan `let nama = "Budi"` dan `let umur = 20`, cetak kalimat:
```
Nama saya Budi, umur 20 tahun
```

## 📤 Output yang Diharapkan
```
22
12
85
3.4
2
289
Nama saya Budi, umur 20 tahun
```

## 💡 Petunjuk
- `%` berguna untuk cek genap/ganjil dan kelipatan (dipakai lagi di Bab Loops).
- Saat `+` bertemu string, JavaScript menggabungkan, bukan menjumlah: `"umur " + 20` ➡️ `"umur 20"`.
