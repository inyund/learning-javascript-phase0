# 📦 Bab 02: Variables & Data Types

Variabel adalah "wadah" untuk menyimpan data, dan setiap data punya **tipe** (`string`, `number`, `boolean`, dll). Di bab ini kamu belajar mengenali tipe data dengan `typeof`, memahami `var`/`let`/`const`, dan membedakan sifat **immutable** vs **mutable**.

## 📌 Urutan Pengerjaan Soal

1. **`02-1-data-type-check.md`** ➡️ Pengecekan tipe data (`typeof`) & nilai *falsy*.
2. **`02-2-mutability-check.md`** ➡️ String *immutability* vs Array *mutability*.
3. **`02-3-tukar-nilai.md`** ➡️ Menukar isi dua variabel (*swap*).
4. **`02-4-var-let-const.md`** ➡️ Perbedaan `var`, `let`, dan `const`.

## 🛠️ Cara Menjawab
Baca file `.md`, lalu tulis kodemu di file `.js` yang bernama sama. Jalankan dengan:
```bash
node 02-1-data-type-check.js
```
