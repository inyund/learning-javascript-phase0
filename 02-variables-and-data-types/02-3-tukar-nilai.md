# 🔀 02-3: Tukar Nilai Dua Variabel (Swap)

## 🎯 Konsep yang Dilatih
Memahami bahwa penugasan (`=`) menimpa nilai lama, sehingga menukar dua variabel butuh **variabel bantu** (*temporary*) agar nilai tidak hilang.

## ✅ Tugas
Diberikan:
```js
let a = 5;
let b = 10;
```
Tukar isinya sehingga `a` menjadi `10` dan `b` menjadi `5`, lalu cetak keduanya.

## 📤 Output yang Diharapkan
```
a = 10
b = 5
```

## 💡 Petunjuk
- Kalau kamu langsung menulis `a = b`, nilai asli `a` keburu hilang tertimpa sebelum sempat dipindah.
- Solusinya butuh satu variabel bantu untuk menyimpan sementara salah satu nilai. Pikirkan urutan langkah yang benar agar tidak ada nilai yang hilang.
