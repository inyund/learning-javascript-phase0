# 🧪 Exercise 1: String Immutability vs Array Mutability

## Problem Statement
Diberikan beberapa variabel String dan Array. Amati apa yang terjadi saat Anda mencoba mengubah karakter indeks String vs elemen Array.

## Starter Code (`mutability-check.js`)
```javascript
const dua = "defaultvaluedua";
dua[2] = "x";
console.log(dua);

const empat = [1, 9, 6];
empat[1] = "valuedilistpertama";
console.log(empat);
```

## Tugas:
Jalankan file `mutability-check.js` dan tulis refleksi analisis Anda pada komentar file.
