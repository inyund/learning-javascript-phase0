# 🚀 Learning JavaScript Phase 0

Selamat datang di repository latihan **JavaScript Phase 0**! Repository ini dirancang khusus dengan urutan kurikulum yang presisi dan bertahap untuk membangun *foundational logic* pemrograman JavaScript dari NOL hingga mahir.

---

## 🗂️ Urutan Kurikulum & Peta Belajar

| Bab | Topik Utama | Deskripsi & Materi |
| :--- | :--- | :--- |
| **01-variables-and-data-types** | **Variabel & Mutabilitas Tipe Data** | Memahami *declaration* (`let`, `const`), pengecekan `typeof`, serta sifat *primitive immutability* (String) vs *reference mutability* (Array). |
| **02-string-manipulation** | **Manipulasi String & Indeks** | Memahami penggabungan string (*concatenation*), akses indeks `str[i]`, metode `substring()`, dan pencarian panjang string (`.length`). |
| **03-conditionals** | **Logika Pengkondisian** | Menguasai percabangan `if-else` bertingkat, nested conditional, serta `switch-case` untuk penanganan nilai acak & range angka. |
| **04-pseudocode-and-algorithms** | **Pseudocode & Alur Logika** | Menerjemahkan permasalahan nyata & formula matematika ke dalam instruksi terstruktur (Pseudocode) sebelum coding. |
| **05-looping** | **Perulangan & Iterasi** | Menguasai perulangan maju-mundur (`for` & `while`), perulangan dengan kondisi modulus, perulangan terpicu, dan *nested loop* (pola bintang). |
| **06-functions** | **Fungsi & Modularisasi Kode** | Membuat fungsi re-usable dengan `parameter` & `return value`, fungsi bertingkat, serta modul konversi data & perbandingan. |

---

## 🛠️ Cara Mengerjakan Latihan
1. **Clone** repository ini ke komputer lokal Anda:
   ```bash
   git clone https://github.com/inyund/learning-javascript-phase0.git
   ```
2. Masuk ke folder repository:
   ```bash
   cd learning-javascript-phase0
   ```
3. Buka folder latihan sesuai urutan bab (`01` s/d `06`).
4. Baca instruksi pada file `.md` di masing-masing folder.
5. Tuliskan jawaban / solusi Anda pada file starter (`.js` / `.txt`) yang sudah disediakan.
6. Uji dan jalankan file Anda menggunakan Node.js di terminal:
   ```bash
   node <nama-file>.js
   ```
