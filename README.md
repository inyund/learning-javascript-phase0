# 🚀 Learning JavaScript Phase 0

Selamat datang di repository latihan **JavaScript Phase 0**! Repository ini berisi kumpulan soal-soal latihan dasar logika pemrograman menggunakan bahasa JavaScript.

Materi di dalam repository ini dirancang bertahap mulai dari perancangan logika Pseudocode, Variabel & Tipe Data, Manipulasi String, Pengkondisian (If-Else & Switch-Case), Perulangan (Looping), hingga Fungsi (Functions).

---

## 🛠️ Cara Mengerjakan
1. **Clone** repository ini ke komputer lokal Anda:
   ```bash
   git clone https://github.com/inyund/learning-javascript-phase0.git
   ```
2. Masuk ke folder repository:
   ```bash
   cd learning-javascript-phase0
   ```
3. Pelajari materi bab secara berurutan (`01` hingga `06`).
4. Tulis jawaban Anda pada file starter (`.js` / `.txt`) di setiap folder latihan.
5. Jalankan kode Anda via Node.js di terminal:
   ```bash
   node <nama-file>.js
   ```
