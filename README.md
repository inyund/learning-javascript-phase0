# 🚀 Learning JavaScript Phase 0

Selamat datang di repository latihan **JavaScript Phase 0**! Repository ini berisi kumpulan soal-soal latihan dasar logika pemrograman menggunakan bahasa JavaScript.

Setiap bab dan file soal di dalam repository ini telah **diurutkan secara presisi (01, 02, 03, ...)** berdasarkan tingkat kesulitan dan alur logika belajar (*learning path*) yang proper.

---

## 🗂️ Peta Kurikulum & Urutan Pengerjaan Soal

### 📌 01-pseudocode-and-algorithms/
1. **`01-1-perhitungan-massa-percepatan`**: Perhitungan fisika/matematika terstruktur.
2. **`01-2-penilaian-grade-siswa`**: Penulisan pseudocode percabangan grade.
3. **`01-3-disney-island-tariff`**: Pseudocode penentuan tarif tiket berdasarkan usia & tinggi.

### 📌 02-variables-and-data-types/
1. **`02-1-data-type-check`**: Pengecekan `typeof` & penanganan nilai falsy.
2. **`02-2-mutability-check`**: Perbedaan String immutability vs Array mutability.

### 📌 03-string-manipulation/
1. **`03-1-play-with-strings`**: Penggabungan kata, indeks `str[i]`, & `substring()`.
2. **`03-2-compare-numbers`**: Pembandingan dua angka dasar.
3. **`03-3-balik-kata`**: Pembalik kata (Reverse String) manual.
4. **`03-4-palindrome-checker`**: Pendeteksi kata palindrome.

### 📌 04-conditionals/
1. **`04-1-grading-system`**: Pengkondisian rentang angka sederhana (Grade A-E).
2. **`04-2-gatcha-system`**: Switch-case penanganan angka acak.
3. **`04-3-bikilipik-diskon`**: Nested if-else role member & bulan lahir.
4. **`04-4-disney-island-tariff`**: Penentuan tarif tiket multi-tier (umur & tinggi).

### 📌 05-looping/
1. **`05-1-ilovecoding`**: Perulangan dasar `for` & `while` (naik/turun).
2. **`05-2-laundry-day`**: Perulangan `while` terpicu.
3. **`05-3-odd-even-factors`**: Step increment & modulus.
4. **`05-4-asterisks-pattern`**: Nested loop pola bintang (persegi & segitiga).
5. **`05-5-three-cols-box`**: Pola grid angka berurutan (`row` x `col`).

### 📌 06-functions/
1. **`06-1-basic-functions`**: Fungsi dasar dengan `parameter`, `return`, & nested call.
2. **`06-2-konversi-menit`**: Modul konversi detik ke format `mm:ss`.
3. **`06-3-xo-counter`**: Pengecekan kesamaan jumlah karakter 'x' dan 'o'.
4. **`06-4-ayo-joget-game`**: Kalkulasi persentase poin & kategori game joget.

---

## 🛠️ Cara Mengerjakan
1. **Clone** repository ini:
   ```bash
   git clone https://github.com/inyund/learning-javascript-phase0.git
   ```
2. Kerjakan soal secara berurutan sesuai nomor awalan file (contoh: kerjakan `02-1-...` sebelum `02-2-...`).
3. Jalankan kode via Node.js di terminal:
   ```bash
   node <nama-file>.js
   ```
