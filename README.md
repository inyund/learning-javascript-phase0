# 🚀 Learning JavaScript Phase 0 (Ultimate Edition)

Selamat datang di repository latihan **JavaScript Phase 0**! Repository ini dirancang secara komprehensif, terstruktur, dan presisi tinggi untuk membangun *problem solving* dan *foundational logic* pemrograman dari dasar hingga siap melangkah ke Phase 1.

Materi di repository ini dikompilasi dari **Instructor VQ** dan dikaya dengan soal-soal latihan **Livecode Simulation (RMT-26 Zurich Fox / Hacktiv8)** seperti Palindrome, Disney Island Tariff, Ayo Joget Game, Asterisks Patterns, hingga Complex Function Modules.

## 🛠️ Cara Mengerjakan
1. **Clone** repository ini ke komputer lokal Anda:
   ```bash
   git clone https://github.com/inyund/learning-javascript-phase0.git
   ```
2. Masuk ke folder repository:
   ```bash
   cd learning-javascript-phase0
   ```
3. Pelajari materi bab secara berurutan (`01` hingga `06`).
4. Tulis jawaban Anda pada file starter (`.js` / `.txt`) di setiap folder latihan.
5. Jalankan kode Anda via Node.js di terminal:
   ```bash
   node <nama-file>.js
   ```
