# 🚀 Learning JavaScript Phase 0

Selamat datang di repository latihan **JavaScript Phase 0**! Repository ini dirancang untuk melatih dasar-dasar logika pemrograman menggunakan bahasa JavaScript, mulai dari Variabel, Tipe Data, Pseudocode, Pengkondisian (If/Else & Switch-Case), Looping, Manipulasi String, hingga Function dasar.

## 🗂️ Struktur Materi & Latihan

- **`01-variables-and-data-types/`**: Memahami mutabilitas variabel, array vs string, dan pengecekan tipe data (`typeof`).
- **`02-pseudocode-and-algorithms/`**: Menulis logika dasar dalam bentuk Pseudocode.
- **`03-conditionals/`**: Logika pengkondisian menggunakan `if-else` bertingkat dan `switch-case`.
- **`04-looping/`**: Perulangan `for`, `while`, increment kustom, dan perulangan bersarang (*nested loop*).
- **`05-string-manipulation/`**: Mengakses karakter indeks, substring, serta algoritma pembalikan kata.
- **`06-functions/`**: Pembuatan fungsi dengan parameter, return value, pemanggilan fungsi bertingkat, dan modul konversi.

## 🛠️ Cara Mengerjakan
1. **Clone** repository ini ke komputer lokal Anda:
   ```bash
   git clone https://github.com/inyund/learning-javascript-phase0.git
   ```
2. Masuk ke folder repository:
   ```bash
   cd learning-javascript-phase0
   ```
3. Buka folder latihan yang ingin dikerjakan.
4. Baca instruksi di file `.md` pada masing-masing folder.
5. Tulis jawaban / kode JavaScript Anda di file starter `.js` atau `.txt` yang telah disediakan.
6. Jalankan file JavaScript Anda menggunakan Node.js untuk mengecek output:
   ```bash
   node <nama-file>.js
   ```
