# 🔢 06-6: Hitung Kemunculan Karakter

## 🎯 Konsep yang Dilatih
Loop untuk menghitung berapa kali sebuah karakter tertentu muncul di dalam string.

## ✅ Tugas
Diberikan `let kata = "mississippi"` dan `let target = "s"`. Hitung berapa kali `target` muncul di dalam `kata`, lalu cetak jumlahnya.

## 📤 Contoh
```js
let kata = "mississippi", target = "s";   // ➡️ 4
let kata = "mississippi", target = "i";   // ➡️ 4
let kata = "banana",      target = "a";   // ➡️ 3
```

## 💡 Petunjuk
- Siapkan satu penghitung, lalu telusuri string karakter demi karakter.
- Setiap kali karakter yang sedang dilihat cocok dengan `target`, naikkan penghitungnya.
