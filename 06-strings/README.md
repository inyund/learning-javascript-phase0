# 🔤 Bab 06: String Manipulation

String adalah deretan karakter. Di bab ini kamu belajar menggabungkan string, mengakses karakter lewat indeks (`str[i]`), mengambil potongan (`substring`), sampai memproses string dengan loop.

> 💡 Semua soal manipulasi di bab ini memakai `for` loop dan `if/else` — konsep yang sudah kamu kuasai di Bab 05 dan 04.

## 📌 Urutan Pengerjaan Soal

1. **`06-1-play-with-strings.md`** ➡️ penggabungan string, indeks `word[i]`, & `substring()`.
2. **`06-2-balik-kata.md`** ➡️ membalik string secara manual (tanpa `.reverse()`).
3. **`06-3-palindrome-checker.md`** ➡️ mendeteksi kata palindrome.
4. **`06-4-hitung-vokal.md`** ➡️ menghitung jumlah huruf vokal.
5. **`06-5-kapital-tiap-kata.md`** ➡️ mengubah awal tiap kata jadi huruf kapital.
6. **`06-6-hitung-karakter.md`** ➡️ menghitung kemunculan sebuah karakter.

## 🛠️ Cara Menjawab
Tulis kodemu di file `.js` yang bernama sama dengan soalnya, lalu jalankan dengan `node <nama-file>.js`.
