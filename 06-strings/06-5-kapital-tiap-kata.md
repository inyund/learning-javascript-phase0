# 🔠 06-5: Kapital Tiap Kata (Title Case)

## 🎯 Konsep yang Dilatih
Memproses string sambil melihat **posisi**: mengubah huruf pertama tiap kata menjadi kapital.

## ✅ Tugas
Diberikan `let kalimat = "belajar javascript itu seru"`. Ubah huruf pertama **setiap kata** menjadi kapital, lalu cetak hasilnya.

## 📤 Output yang Diharapkan
```
Belajar Javascript Itu Seru
```

## 💡 Petunjuk
- Kunci soal: bagaimana caramu tahu sebuah huruf adalah **awal kata**? Perhatikan posisinya dan karakter di sebelumnya.
- Untuk mengubah satu huruf jadi kapital ada method `.toUpperCase()`.
- Bangun string hasil sepotong demi sepotong sambil menelusuri kalimat.
