# 游댃 Exercise 3: Palindrome Checker

## Objective
Mengecek apakah sebuah kata merupakan **Palindrome** (kata yang jika dibalik tetap memiliki susunan huruf yang sama persis, misal: `katak`, `kodok`, `racecar`, `kasur rusak`).

## Problem Statement
Diberikan variabel `kata`. Buatlah logika untuk menguji apakah `kata` tersebut palindrome:
- Jika Palindrome 俱뫮잺 Cetak `true`
- Jika Bukan Palindrome 俱뫮잺 Cetak `false`

### Example Test Cases:
- `kata = 'katak'` 俱뫮잺 `true`
- `kata = 'blanket'` 俱뫮잺 `false`
- `kata = 'civic'` 俱뫮잺 `true`
