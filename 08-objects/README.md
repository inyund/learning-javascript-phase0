# 🗃️ Bab 08: Objects

Object menyimpan data sebagai pasangan **key–value** (misal `nama: "Andi"`), cocok untuk mewakili satu "benda" dengan banyak properti. Bab ini melatih dari dasar object sampai mengolah **array of objects** — kombinasi array (Bab 07) dan object yang sangat sering dipakai di dunia nyata.

## 📌 Urutan Pengerjaan Soal

1. **`08-1-object-basics.md`** ➡️ membuat, mengakses (`.key` & `["key"]`), menambah, mengubah properti.
2. **`08-2-nested-object.md`** ➡️ mengakses object & array di dalam object.
3. **`08-3-iterasi-object.md`** ➡️ menelusuri seluruh key–value dengan `for...in`.
4. **`08-4-array-of-objects.md`** ➡️ menelusuri daftar object.
5. **`08-5-total-nilai.md`** ➡️ menjumlah & merata-rata sebuah properti dari daftar object.
6. **`08-6-cari-tertinggi.md`** ➡️ mencari object dengan nilai properti tertinggi.

> 💡 Semua soal dikerjakan **inline** dengan loop & percabangan. Membungkusnya jadi fungsi kamu pelajari di Bab 09.

## 🛠️ Cara Menjawab
Tulis kodemu di file `.js` yang bernama sama dengan soalnya, lalu jalankan dengan `node <nama-file>.js`.
