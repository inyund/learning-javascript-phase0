# 📦 Exercise 2: Three Cols Box (Pola Angka Grid)

## Problem Statement
Buatlah program perulangan dengan variabel `row` yang menentukan jumlah baris. Setiap baris memiliki 3 kolom (`col`).
Cetak angka berurutan mulai dari 1 sampai nilai maksimum baris x kolom.

### Example:
```javascript
let row = 3;
// Output:
// 1 2 3
// 4 5 6
// 7 8 9
```
