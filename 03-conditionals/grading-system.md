# 📊 Challenge: Grading System

## Objective
Menerapkan pengkondisian range angka menggunakan `switch (true)` atau `if-else`.

## Problem Statement
Diberikan variabel `nama` dan `nilai`. Tentukan nilai grade sesuai dengan rentang berikut:
- 80 - 100: Grade "A"
- 65 - 79: Grade "B"
- 50 - 64: Grade "C"
- 35 - 49: Grade "D"
- 0 - 34: Grade "E"
- Selain itu: "Nilai Invalid"

Tampilkan hasil berupa huruf Grade ke layar.
