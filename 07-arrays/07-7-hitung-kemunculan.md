# 🔢 07-7: Hitung Kemunculan Nilai

## 🎯 Konsep yang Dilatih
Menghitung berapa kali sebuah nilai muncul di dalam array — mirip soal hitung karakter di string, tapi pada elemen array.

## ✅ Tugas
Diberikan `let angka = [3, 1, 3, 2, 3, 1]` dan `let target = 3`. Hitung berapa kali `target` muncul, lalu cetak jumlahnya.

## 📤 Contoh
```js
let target = 3;   // ➡️ 3
let target = 1;   // ➡️ 2
let target = 9;   // ➡️ 0
```

## 💡 Petunjuk
- Pakai penghitung dan telusuri array; naikkan penghitung setiap kali elemen sama dengan `target`.
- Bandingkan dengan `===` agar tipe & nilainya benar-benar sama (ingat Bab 03).
