# 🔗 07-9: Gabung Dua Array

## 🎯 Konsep yang Dilatih
Menyalin isi beberapa array ke dalam satu array hasil dengan loop — logika di balik `.concat()` / spread, dikerjakan manual.

## ✅ Tugas
Diberikan `let a = [1, 2, 3]` dan `let b = [4, 5, 6]`. Gabungkan keduanya menjadi satu array `[1, 2, 3, 4, 5, 6]` secara manual, lalu cetak.

## 📤 Output yang Diharapkan
```
[ 1, 2, 3, 4, 5, 6 ]
```

## 💡 Petunjuk
- Mulai dari array hasil kosong.
- Telusuri array `a` dan `push` tiap elemennya ke hasil, lalu lakukan hal yang sama untuk `b`.
- Latih tanpa `.concat()` dulu supaya paham prosesnya.
