# 🔄 07-6: Balik Array (Manual)

## 🎯 Konsep yang Dilatih
Membangun array baru dengan menelusuri array asal dari **belakang ke depan** — tanpa memakai built-in `.reverse()`.

## ✅ Tugas
Diberikan `let angka = [1, 2, 3, 4, 5]`. Balik urutannya menjadi `[5, 4, 3, 2, 1]` secara manual, lalu cetak.

## 📤 Output yang Diharapkan
```
[ 5, 4, 3, 2, 1 ]
```

## 💡 Petunjuk
- Ingat teknik "balik kata" di Bab 06 — di sini polanya sama, tapi yang dibalik elemen array, bukan huruf.
- Mulai dari array kosong, lalu isi dengan menelusuri array asal dari indeks terakhir menuju `0`.
- Dilarang pakai `.reverse()`; justru loop mundurnya yang jadi latihan.
