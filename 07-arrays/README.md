# 📚 Bab 07: Arrays

Array adalah **daftar berurutan** yang menyimpan banyak nilai dalam satu variabel, diakses lewat indeks (mulai dari `0`). Bab ini paling banyak soalnya karena array dipakai di mana-mana — kamu berlatih dari dasar sampai operasi seperti mencari, membalik, dan menghapus duplikat.

## 📌 Urutan Pengerjaan Soal

1. **`07-1-array-basics.md`** ➡️ membuat, mengakses indeks, `.length`, `.push`, mengubah elemen.
2. **`07-2-jumlah-max-min.md`** ➡️ menelusuri array untuk hitung total, terbesar, terkecil.
3. **`07-3-filter-manual.md`** ➡️ menyaring elemen ke dalam array baru.
4. **`07-4-rata-rata.md`** ➡️ menghitung nilai rata-rata dari isi array.
5. **`07-5-cari-dan-index.md`** ➡️ mencari nilai & posisinya (indeks).
6. **`07-6-balik-array.md`** ➡️ membalik urutan array secara manual.
7. **`07-7-hitung-kemunculan.md`** ➡️ menghitung berapa kali sebuah nilai muncul.
8. **`07-8-hapus-duplikat.md`** ➡️ membuat array unik tanpa nilai kembar.
9. **`07-9-gabung-array.md`** ➡️ menggabungkan dua array secara manual.
10. **`07-10-kata-terpanjang.md`** ➡️ mencari elemen terpanjang di array string.

> 💡 Semua soal di sini dikerjakan **inline** dengan loop & percabangan (Bab 04–05). Membungkusnya jadi fungsi baru kamu pelajari di Bab 09.

## 🛠️ Cara Menjawab
Tulis kodemu di file `.js` yang bernama sama dengan soalnya, lalu jalankan dengan `node <nama-file>.js`.
