# ➗ 07-4: Rata-rata Isi Array

## 🎯 Konsep yang Dilatih
Menggabungkan akumulasi total dengan `.length` untuk menghitung rata-rata sebuah array angka.

## ✅ Tugas
Diberikan `let nilai = [80, 90, 70, 100, 60]`. Hitung rata-rata semua elemennya, lalu cetak.

## 📤 Output yang Diharapkan
```
Rata-rata: 80
```

## 💡 Petunjuk
- Jumlahkan dulu semua elemen (teknik akumulasi seperti di soal `07-2`).
- Rata-rata = total dibagi banyaknya elemen. Banyak elemen bisa didapat dari `.length`, bukan diangka-hardcode.
