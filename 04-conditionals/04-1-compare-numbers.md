# ⚖️ 03-1: Bandingkan Angka

## 🎯 Konsep yang Dilatih
Percabangan `if / else if / else` untuk membandingkan dua nilai.

## ✅ Tugas
Diberikan dua variabel `angka1` dan `angka2`. Bandingkan keduanya dan cetak hasilnya sesuai aturan berikut:

| Kondisi | Output |
|---|---|
| `angka2` lebih besar dari `angka1` | `true` |
| `angka1` lebih besar dari `angka2` | `false` |
| `angka1` sama dengan `angka2` | `"-1"` |

## 📤 Contoh
```js
let angka1 = 5, angka2 = 3;   // ➡️ false   (angka1 lebih besar)
let angka1 = 3, angka2 = 5;   // ➡️ true    (angka2 lebih besar)
let angka1 = 4, angka2 = 4;   // ➡️ -1      (sama)
```

## 💡 Petunjuk
Perhatikan baik-baik: output `true` diberikan ketika **angka2** yang lebih besar, bukan angka1. Baca tabel dengan teliti.
