# 🔀 Bab 04: Conditionals

Percabangan membuat program bisa mengambil keputusan. Di bab ini kamu berlatih dari `if/else` sederhana, `switch-case`, `if` bersarang (*nested*), sampai bentuk singkatnya (ternary). Semua memanfaatkan operator perbandingan & logika dari Bab 03.

## 📌 Urutan Pengerjaan Soal

1. **`04-1-compare-numbers.md`** ➡️ `if/else` dasar membandingkan dua angka.
2. **`04-2-grading-system.md`** ➡️ percabangan rentang nilai (Grade A–E).
3. **`04-3-gacha-system.md`** ➡️ `switch-case` untuk angka acak 1–5.
4. **`04-4-bikilipik-diskon.md`** ➡️ *nested* `if/else` (role member & bulan lahir).
5. **`04-5-disney-island-tariff.md`** ➡️ percabangan syarat ganda (umur & tinggi).
6. **`04-6-bmi-category.md`** ➡️ klasifikasi rentang hasil hitungan (BMI).
7. **`04-7-batu-gunting-kertas.md`** ➡️ *nested* `if/else` menentukan pemenang.
8. **`04-8-ternary-operator.md`** ➡️ bentuk singkat `if/else` satu baris.

> 🔁 Soal `04-5` adalah versi JavaScript dari pseudocode yang kamu tulis di `01-3`.

## 🛠️ Cara Menjawab
Tulis kodemu di file `.js` yang bernama sama dengan soalnya, lalu jalankan dengan `node <nama-file>.js`.
