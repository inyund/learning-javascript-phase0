# 🎯 Bab 10: Combined Challenges

Bab penutup Phase 0. Tiap soal di sini adalah tantangan gaya *live code* yang menggabungkan hampir semua materi sebelumnya: **fungsi**, **loop**, **percabangan**, **string**, **array**, dan **object** sekaligus. Kerjakan setelah kamu nyaman dengan Bab 01–09.

## 📌 Urutan Pengerjaan Soal

1. **`10-1-summary-pengunjung.md`** ➡️ mengolah array multidimensi menjadi laporan.
2. **`10-2-relationship-report.md`** ➡️ mengurai array string & meringkasnya jadi laporan.
3. **`10-3-gotta-group-em-all.md`** ➡️ mengelompokkan array of objects & mencari yang terkuat.
4. **`10-4-ongkos-kirim.md`** ➡️ program modular dengan beberapa fungsi bantu.

## 🛠️ Cara Menjawab
Tulis kodemu di file `.js` yang bernama sama dengan soalnya, lalu jalankan dengan `node <nama-file>.js`.
