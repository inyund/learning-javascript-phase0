# 📐 Bab 01: Pseudocode & Algorithms

Sebelum menulis kode JavaScript, kita belajar menuangkan logika dalam bentuk **pseudocode** — "kode pura-pura" berbahasa manusia yang menjelaskan langkah-langkah algoritma tanpa terikat sintaks bahasa pemrograman.

## 📖 Kosakata Pseudocode

Gunakan kosakata berikut (konsisten dengan materi kelas):

| Perintah | Arti |
|---|---|
| `STORE "x" WITH VALUE 10` | Buat variabel `x` berisi 10 |
| `STORE "x" WITHOUT ANY VALUE` | Buat variabel `x` kosong |
| `CALCULATE "a" TIMES "b"` | Hitung `a × b` (juga: `PLUS`, `MINUS`, `DIVIDED BY`) |
| `SET "x" VALUE WITH ...` | Isi/ubah nilai variabel `x` |
| `IF ... THEN ...` / `ELSE IF ...` / `ELSE ...` | Percabangan |
| `DISPLAY "x"` | Tampilkan isi variabel `x` |
