# 📐 Bab 01: Pseudocode & Algorithms

## 🎯 Objective
1. Melatih *computational thinking* sebelum menulis baris kode pemrograman.
2. Menerjemahkan masalah logika verbal ke dalam format instruksi terstruktur **Pseudocode** (`STORE`, `CALCULATE`, `IF-THEN-ELSE`, `DISPLAY`).

---

## 📂 Latihan:
Tuliskan penyelesaian Pseudocode untuk 3 kasus latihan berikut pada file `solution.txt`:
1. **Perhitungan Massa x Percepatan**: Rumus fisika sederhana.
2. **System Penilaian Grade**: Klasifikasi nilai 0-100 menjadi Grade A-E.
3. **Disney Island Tariff**: Penentuan tiket berdasarkan usia & tinggi anak.
