# 📐 Pseudocode & Basic Algorithms

## Objective
Mampu menerjemahkan problem matematika dasar dan logika pengkondisian ke dalam format **Pseudocode**.

## Problem Statement

### Soal 1: Perhitungan Fisika/Matematika
Buatlah pseudocode untuk menghitung nilai `hasil` dari perkalian massa (`m` = 600) dan percepatan (`a` = 2), lalu tampilkan nilainya.

### Soal 2: Klasifikasi Grade Nilai
Buatlah pseudocode untuk menentukan grade siswa berdasarkan nilai yang didapatkan dengan ketentuan:
- Nilai 80 - 100: Grade "A"
- Nilai 65 - 79: Grade "B"
- Nilai 50 - 64: Grade "C"
- Nilai 35 - 49: Grade "D"
- Nilai 0 - 34: Grade "E"
- Di luar 0 - 100: "Nilai Invalid"
Tampilkan nama dan grade siswa tersebut.

## Starter File (`solution.txt`)
Tuliskan Pseudocode Anda pada file `solution.txt`.
